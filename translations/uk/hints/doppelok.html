Ітеруватись по списку з конкретним кроком нам допоможуть дві функції range() і len(). Функція range() може приймати як аргументи (початок відрахунку, кінець, крок). Початок ми обираємо 0, кінець буде довжина списку (тобто використаємо функцію len()), а крок це буде наш size. Також зміниться початок генератору, він тепер буде додавати елементи по індексу в списку. 

a = [items[i] for i in range(0, len(items), size)]

Нічого не працює. SOS

Нам потрібно додати фінальний апгрейд нашого генератору списку. Зараз ми додаємо до списку тільки перший, четвертий та сьомий елементи (пам'ятай, що нумерація починається з 0, а не з 1). Тому ми допишемо код, що будемо додавати елементи з - по.

from collections.abc import Iterable


def chunking_by(items: list[int], size: int) -> Iterable[list[int]]:

    return [items[i:i+size] for i in range(0, len(items), size)]

Добре, добре! Оце настрій!=) Варіант рішення місії за допомогою циклу for та виразу yield. Така функція теж сама по собі вважається генератором. 

from collections.abc import Iterable


def chunking_by(items: list[int], size: int) -> Iterable[list[int]]:

    for i in range(0, len(items), size):
        yield items[i:i + size]
