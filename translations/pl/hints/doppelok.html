Nie wiem, jak zacząć rozwiązywać tę misję.

Chodzi o to, że musi się Pan upewnić, że przechodzi Pan przez wiersz w określonym kroku. Na przykład, można użyć pętli lub list comprehensions.

Potrzebuję pomocy, aby kontynuować misję.

W naszym rozwiązaniu użyjemy generatora list i kilku funkcji. len() (example), range() (example), list comprehensions.

Proszę o pomoc!

Napiszmy zwykłe rozumienie listy, które zmodyfikujemy. W tym zrozumieniu listy będziemy po prostu iterować po podanej liście. 

a = [i for i in items]

Nie wiem już, co robić. Potrzebuję małej podpowiedzi.

Dwie funkcje range() i len() pomogą nam iterować po liście z określonym krokiem. Funkcja range() może przyjąć jako argumenty (licząc początek, koniec, krok). Wybieramy 0 jako początek, koniec będzie długością listy (czyli użyjemy funkcji len() ), a krok będzie naszym size. Początek rozumienia listy również się zmieni, będzie teraz dodawać elementy według indeksu na liście. 

a = [items[i] for i in range(0, len(items), size)]

Nic nie działa. SOS

Musimy dodać ostatnią aktualizację do naszego rozumienia listy. Teraz dodajemy do listy tylko pierwszy, czwarty i siódmy element (proszę pamiętać, że numeracja zaczyna się od 0, a nie od 1). Dlatego dodamy kod, który doda elementy od - do. 

a = [items[i:i+size] for i in range(0, len(items), size)]

Nie chcę się poddawać, proszę mi pomóc!

Wszystko, co musi Pan zrobić, to zawinąć kod w wyrażenie funkcji i zwrócić nasze rozumienie listy. 

from collections.abc import Iterable


def chunking_by(items: list[int], size: int) -> Iterable[list[int]]:

    return [items[i:i+size] for i in range(0, len(items), size)]

Dobra, dobra! That's the spirit!=) Opcja rozwiązania misji za pomocą pętli for i wyrażenia yield. Taka funkcja jest również uważana za generator sam w sobie. 

from collections.abc import Iterable


def chunking_by(items: list[int], size: int) -> Iterable[list[int]]:

    for i in range(0, len(items), size):
        yield items[i:i + size]
